# HR-Analytics-Employee-Performance-Report

## Project Overview
This project focuses on analyzing employee performance and attrition trends using **Excel**.  
The goal is to help HR teams understand key metrics related to employee satisfaction, performance, and retention.

## Tools & Techniques
- **Microsoft Excel**
  - Data Cleaning  
  - Excel Functions (IF, VLOOKUP, AVERAGEIF, COUNTIF, etc.)  
  - Pivot Tables for summarizing data  
  - Charts for visualization (Bar, Pie, Line charts)
    
## 🧹 Data Cleaning Steps
- Removed duplicate and null records  
- Corrected data types (date, numeric values)  
  
## 🧮 Excel Functions Used
=IF(S2>5000,"High Sales",IF(S2>2000,"Medium Sales","Low Sales")) ) → Classify performance level

## Key Takeaways
Learned to handle missing and inconsistent data (e.g., blank postal codes) using Excel functions like IF, TEXT.

Used PivotTables, IF formulas to calculate key performance indicators such as Total Sales and Average Discount across multiple regions and product categories. Built formulas to classify performance levels (e.g., High/Medium/Low Sales) for better business insights.

Learned effective color combinations (Teal + Light Gray + Orange) and layout structuring for professional presentation and easy interpretation.
